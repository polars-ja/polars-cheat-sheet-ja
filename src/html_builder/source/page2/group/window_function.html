{{ define "group_window_function" }}

ウィンドウ関数を使用すると、 複数の列で同時にグループ化することができます。 

df.select(
    [
        "names",
        "groups",
        pl.col("random").sum().over("names") \
          .alias("sum_by_names"),
        pl.col("random").sum().over("groups") \
          .alias("sum_by_groups"),
    ]
)
